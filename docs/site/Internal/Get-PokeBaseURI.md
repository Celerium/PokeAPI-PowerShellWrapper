# Get-PokeBaseURI

## SYNOPSIS
Shows the Poke base URI global variable.

## SYNTAX

```powershell
Get-PokeBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-PokeBaseURI cmdlet shows the Poke base URI global variable value.

## EXAMPLES

### EXAMPLE 1
